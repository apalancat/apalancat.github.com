Paymill Subscriptions in Ruby on Rails

Dec 3rd, 2012

Paymill is a Stripe clone available for those of us in Europe. There’s some controversy with the cloning side of things, but either way this is a long awaited opportunity for web businesses here in the Old Continent.

paymill-ruby (github) is a gem for communicating with paymill’s API. Right now it’s in a very early stage, but it works fine for the basics. To install it: gem install paymill

In order to create subscriptions the steps are simple, the only thing needed is a paymill test account.

The paymill control panel lets you create Offers, which are how they call Subscription Plans. Once created, the plans can be used to subscribe clients via the API. The names beginning with ‘offer_’ are the plan IDs that will be needed to setup the subscription with a client and credit card.

Check out the sample app if you want to give it a try.
